## 계속 도전해야 하는 이유

<div class="formula-box">
  <span class="dim">일반 선정률</span> <span class="accent">3~10%</span> <span class="dim">| 유명 영화제</span> <span class="accent">1~2%</span><br>
  <span class="dim">리젝이 정상이다</span>
</div>

- **리젝 이유**: 영화 품질 외에도 프로그래밍 균형, 심사위원 취향, 테마 시의성 등 통제 불가 변수
- 30곳 이상 출품하면 자신의 영화와 맞는 영화제를 만날 확률이 올라감
- 첫 번째 선정이 연쇄 반응을 일으킴 (월계관 → 신뢰도 → 추가 선정)

```mermaid
flowchart LR
    A["🎬 출품<br>30곳+"] --> B["🏅 첫 선정"]
    B --> C["🌿 월계관<br>신뢰도 ↑"]
    C --> D["📈 추가 선정<br>연쇄 반응"]

    style A fill:#1a1926,stroke:#5c5c66,color:#ededef
    style B fill:#1a1926,stroke:#8b5cf6,color:#ededef

```
